Sobre EJS

O que é?

O “E” é algo variável, pode ser “Embedded”, assim como “Effective”, “Elegant” ou até “Easy”. EJS é uma linguagem de modelagem simples, que gera código HTML com simples marcação de JavaScript. Sem muito rigor para organizar as coisas. Sem reinventar iteração ou controle de fluxo. É simplesmente JavaScript.

Como usar?

Nesta aplicação o ejs é renderizado através do template (EJS), passagem dos dados e das opções de configuração via JSON.

Recursos:

  • Fluxo de controle com <% %>
  • Exibir saída com <%= %>
  • Pular linha com -%> no final da tag
  • Espaço em branco para controlar o fluxo com <%_ _%>
  • Delimitadores customizados com (e.g. [? ?] ao invés de <% %>
  • Includes

Para saber mais sobre a estrutura deste template recomendo ler em EJS NPM.
